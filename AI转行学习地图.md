
### 概念 2：爬虫是什么？
```
爬虫 = 帮你自动抓网页数据的小工具
以前：手动打开 5 个网站 + 抄数据 = 1 小时
现在：跑一下爬虫 = 5 秒
```

### 概念 3：为什么先学爬虫不直接学 AI？
```
完整 AI 项目 = 抓数据 + 让 AI 处理 + 输出结果
爬虫 = 完整链路的第一步
不学爬虫 = 后面所有 AI 项目都做不出来
```

```
不是你发明 AI
```
